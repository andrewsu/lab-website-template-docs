# Icon

```liquid
{%
  include icon.html
  icon="fa-solid fa-heart"
%}
```

| Parameter | Description                                                                                                                            |
| --------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| `icon`    | Full class name of [Font Awesome free icon](https://fontawesome.com/search?o=r\&m=free), or name of a custom icon SVG in `/_includes`. |
